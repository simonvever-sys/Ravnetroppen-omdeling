# Supabase setup (simpel version)

## 1) Koer SQL
- Opret et Supabase projekt
- Gaa til SQL Editor
- Koer `SUPABASE_SETUP.sql`

## 2) Indsaet nøgler
Aabn `supabaseClient.js` og saet:
- `SUPABASE_URL`
- `SUPABASE_ANON_KEY`

## 3) Deploy
- Upload til GitHub Pages
- Login i appen med:
  - `admin / admin`
  - `1-20 / ravnetroppen`

Det er det. Ingen ekstra Auth-brugere er noedvendige i denne version.
